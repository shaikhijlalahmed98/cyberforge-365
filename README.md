## ⚙️ Daily Workflow

Every morning at 7:00 AM, my Claude routine fires and delivers a 5-part briefing:

1. **Today's Briefing** — Day X of 365, theme, connection to yesterday
2. **Core Concept** (60%) — Explained for a backend dev with zero security background
3. **Real-World Incident** (20%) — A documented breach tied to today's concept
4. **Hands-On Lab** (15%) — 30–60 min practical exercise
5. **Reflection & Tomorrow** (5%) — Journal prompt + progress.md update line

**Special days:**
- 🔄 Every 7th day → Week recap & quiz
- 🏆 Every 30th day → Monthly capstone (portfolio-grade artifact)
- 🎯 Every 90th day → Quarterly assessment + public publication
- 🚨 1–2x per month → Red Team Roulette (surprise breach simulation)
