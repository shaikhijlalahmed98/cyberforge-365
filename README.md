# CyberForge 365

Mission

To build a complete, practical, and publishable foundation in cybersecurity in 365 days by combining daily learning, hands-on labs, and public-facing case studies.

Goals

- Learn foundational and advanced cybersecurity topics through a structured 365-day curriculum.
- Complete weekly hands-on labs that demonstrate applied skills.
- Produce a public portfolio with case studies and architecture diagrams for each major project.
- Track daily progress and reflections to maintain momentum and accountability.

Accountability

- Daily: append entries to progress.md and notes/day-###.md with what you studied and what you built.
- Weekly: complete at least one lab in the labs/ folder and add a summary to portfolio/ when a project is complete.
- Public commitment: publish milestone case studies in the portfolio/ folder and link them from README.
