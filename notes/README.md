# Notes — day-by-day notes for CyberForge 365

How to use this folder

- Create one file per day using a consistent name: day-001.md, day-002.md, etc.
- Each day file should contain a short reflection, commands run, snippets, and links to any resources used.

Day file template

- Date: YYYY-MM-DD
- Day #: n
- Objectives:
- Commands / snippets:
- Notes / observations:
- Links / references:
- Time spent:

Example: create notes/day-001.md for Day 1.
