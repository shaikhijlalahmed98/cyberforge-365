# CyberForge 365 — 365-day Master Plan

This is the master curriculum for the 365-day learning plan. Claude (or another assistant) will generate the daily breakdown on the first session and append it here.

Structure

- Daily entries: each day will have a focused learning objective, readings, hands-on task, and a mini-assessment or self-check.
- Weekly themes: each 7-day block focuses on a single topic area (e.g., Linux fundamentals, Networking, Web Application Security, Offensive tooling, Threat Hunting, Cloud security, DevSecOps, Blue Team, Purple Team exercises, Incident Response, Red Teaming, etc.).
- Milestones: every 30 days there will be a milestone project or mini-certification target.

How to use this file
