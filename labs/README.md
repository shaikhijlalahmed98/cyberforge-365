# Labs — hands-on lab work for CyberForge 365

How to structure labs

- Organize labs by week: labs/week-01/, labs/week-02/, ...
- Each lab should include a README.md with objectives, setup instructions, expected results, and a short assessment.
- Keep lab artifacts (scripts, configs, diagrams) in the same week folder.

Example layout

- labs/week-01/README.md — lab overview
- labs/week-01/vm-setup.md — VM and environment setup
- labs/week-01/step-1.md — steps and verification

Add completed lab summaries to the portfolio/ folder when they mature into case studies.
