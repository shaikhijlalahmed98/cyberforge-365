## Distribution channels

Each artifact gets published to **at least 3 channels**:

1. **GitHub** (this repo + standalone if appropriate)
2. **LinkedIn article** (long-form, professional audience)
3. **dev.to / Medium / personal blog** (technical audience)
4. **Twitter/X thread** (#CyberForge365)
5. **Hacker News / Reddit r/netsec** (if quality merits)

## Quality bar

Each portfolio piece must:
- Solve a real problem (even if for a fictional system)
- Cite real incidents to ground it
- Include diagrams (not just text walls)
- Have an "executive summary" at the top — busy hiring managers need this
- Show my unique voice — not generic Claude output
