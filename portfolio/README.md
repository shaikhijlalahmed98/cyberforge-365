# Portfolio — case studies & architecture diagrams for CyberForge 365

Purpose

- Collect polished write-ups, architecture diagrams, post-mortems, and case studies suitable for sharing publicly.

How to add items

- For each major project or milestone, create a folder with a README.md that contains the case study, diagrams (SVG/PNG), threat models, and reproductions of lab results.
- Link notable portfolio items from the top-level README.

Example

- portfolio/2026-06-networking-case-study/README.md
- portfolio/2026-06-networking-case-study/diagram-architecture.svg

When ready, export diagrams to a public doc or GitHub Pages site.
